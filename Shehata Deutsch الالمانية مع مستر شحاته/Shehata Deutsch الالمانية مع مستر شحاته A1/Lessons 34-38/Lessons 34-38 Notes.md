## Folge 36: Was ist Nominativ? – الفاعل في اللغة الألمانية

### 📚 Grammatik: Nominativ & Akkusativ Tabelle

|Fall|Maskulin|Neutrum|Feminin|Plural|
|---|---|---|---|---|
|**Nominativ**|der / ein / kein|das / ein / kein|die / eine / keine|die / — / keine|
|**Akkusativ**|**den / einen / keinen**|das / ein / kein|die / eine / keine|die / — / keine|

> ⚠️ **Nur Maskulin ändert sich im Akkusativ:** der → den, ein → einen, kein → keinen

---

### 📚 Nominativ-Regeln

- **Wer?** → Person als Subjekt
- **Was?** → Sache als Subjekt
- Nach **sein / werden / bleiben / heißen** → immer Nominativ

**Demonstrativpronomen (Nominativ):**

|Genus|Artikel|Demonstrativ|
|---|---|---|
|Maskulin|der|dieser|
|Neutrum|das|dieses|
|Feminin|die|diese|
|Plural|die|diese|

## Folge 37: Der Akkusativ (1) – المفعول به

### 📚 Grammatik: Nominativ vs. Akkusativ

| |Maskulin|Neutrum|Feminin|Plural|
|---|---|---|---|---|
|**Nominativ**|der / ein / kein|das / ein / kein|die / eine / keine|die / — / keine|
|**Akkusativ**|**den / einen / keinen**|das / ein / kein|die / eine / keine|die / — / keine|

**Frage im Nominativ:** Wer? Was?  
**Frage im Akkusativ:** Wen? Was?

**Beispiele:**

- Der Lehrer fragt **die Schüler**. → Wen fragt der Lehrer? → **die Schüler** (Akk.)
- Richard kauft **den Stuhl**. → Was kauft Richard? → **den Stuhl** (Akk.)

## Folge 38: Der Akkusativ (2) – المفعول به

### 📚 Grammatik: Wortstellung

Im Deutschen kann das Objekt (Akkusativ) am Satzanfang stehen → der Artikel bleibt **den / einen / keinen** (Mask.)

**Beispiele:**

- Den Mann sieht die Frau nicht.
- Den Onkel begrüßt der Junge.

**Verben + Akkusativ (transitive Verben):**  
kaufen – besuchen – fragen – brauchen – sehen – hören – lesen – nehmen – trinken – kennen – lieben

---
