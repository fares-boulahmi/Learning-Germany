L40: Possessivpronomen im Akkusativ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
RULES:
- ONLY masculine (der) nouns change in accusative: possessive + -en (meinen, deinen, seinen…)
- Neuter, feminine, and plural forms are IDENTICAL to nominative

VOCABULARY:
der Regenschirm — umbrella | der Kuli — pen | der Koffer — suitcase | der Schlüssel — key | der Rucksack — backpack

TRAPS: ⚠️ CONNECTS TO ACTIVE ERROR
- Writing "sein Koffer" instead of "seinen Koffer" (mask. noun → accusative needs -en)
- Using "ihr" for subject-referencing possessives wrong: "Ihren Vater freut sich" → WRONG; nominative → "Ihr Vater"
- Mixing up ihr (her) vs. ihr (their) vs. Ihr (formal Sie) possessive agreement

L41: Verben mit Akkusativ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
RULES:
- These verbs ALWAYS demand an accusative object: suchen, brauchen, lieben, besuchen, vergessen, nehmen, hören, sehen, waschen, heiraten, kochen, fragen, finden, lernen, lesen, schreiben, trinken, essen
- Negation uses kein- (not nicht): "ich habe keinen Bruder / kein Geld / keine Schwester"

TRAPS:
- Using "nicht" instead of "kein" before nouns: "ich mag nicht Sport" → "ich mag keinen Sport"
- Forgetting accusative masculine ending with kein: "kein Kaffee" → "keinen Kaffee" (der Kaffee)

L42: Präpositionen mit Akkusativ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
RULES:
- Fixed accusative prepositions: für · durch · gegen · um · bis · ohne · wider · entlang
- "entlang" FOLLOWS the noun: "die Straße entlang" (not "entlang die Straße")
- "gegen" = against / around (time); "um" = around (movement/orbit) / at (exact time)

VOCABULARY:
gegen — against/around | durch — through/by | ohne — without | bis — until/up to | entlang — along | wider — against (formal)

TRAPS:
- Confusing gegen (approximate time: "gegen 11 Uhr") vs. um (exact time: "um 11 Uhr")
- Forgetting accusative after these prepositions: "durch Europa" ✅ but noun gender still matters for articles

L43: Essen, mögen, gerne, lieber
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
RULES:
- mögen conjugates irregularly: mag / magst / mag / mögen / mögt / mögen
- gern = enjoy doing/having; lieber = prefer (comparative); pattern: "Ich esse gern X → ich mag kein Y → ich esse lieber X"
- "lieber" does NOT combine with "kein" — "ich trinke lieber keinen Kaffee" is wrong; use "ich trinke lieber Tee" OR "ich mag keinen Kaffee"

VOCABULARY:
das Frühstück — breakfast | das Mittagessen — lunch | das Abendessen — dinner | frühstücken — to have breakfast | möchten — would like (polite form)

TRAPS: ⚠️ CONNECTS TO ACTIVE ERROR
- "lieber" + "kein" together is redundant/wrong — pick one negation strategy
- "ich trinke gern kein Kaffee" — also wrong; gern negates positively, kein negates the noun
- "wir essen lieben Pizza" → "wir essen lieber Pizza" (lieber, not lieben)

L44: Die starken Verben
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
RULES:
- Vowel changes only in du / er,sie,es forms; all other persons = infinitive stem
- e → i: essen (isst), nehmen (nimmt), sprechen (spricht), geben (gibt), helfen (hilft), vergessen (vergisst)
- e → ie: sehen (sieht), lesen (liest), empfehlen (empfiehlt)
- a → ä: fahren (fährt), schlafen (schläft), tragen (trägt), waschen (wäscht), laufen (läuft)

CONJUGATION NOTE:
wissen: ich weiß / du weißt / er weiß (irregular — ß in sg.)
werden: ich werde / du wirst / er wird
lassen: ich lasse / du lässt / er lässt

TRAPS:
- Applying vowel change to wir/ihr/Sie forms: "wir fähren" → WRONG → "wir fahren"
- a → ä verbs DO NOT add umlaut in du-imperative: "fahr! / lauf! / schlaf!" NOT "fähr!"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Hauptgerichte: Hähnchenbrustfilet, Rindersteak, Brathähnchen, Schweinebraten
Beilagen: Pommes Frites, Bratkartoffeln, Reis, Spinat
Getränke: Saft, Tee, Kaffee, Bier, Wein
Restaurant phrases: Ich hätte gern… / Ich nehme… / Was bekommen Sie? / Hat es geschmeckt?

TRAPS:
- "bringen uns Sie" → WRONG word order → "bringen Sie uns"
- "Nehmen bitte Platz" → WRONG → "Nehmen Sie bitte Platz" (Sie must follow verb in imperative)

L46: Der Imperativ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
RULES:
- du-form: remove -st from du-present | ihr-form: same as present | Sie-form: Verb + Sie
- e→i/ie strong verbs use changed vowel in du, NO -e ending: sprich! / iss! / nimm! / lies!
- a→ä strong verbs use UNCHANGED vowel in du: fahr! / lauf! / schlaf! (NOT fähr!)
- Separable verbs: prefix goes to end → "Kauf ein! / Mach das Licht aus!"
- sein: Sei / Seid / Seien Sie | haben: Hab / Habt / Haben Sie

TRAPS: ⚠️ CONNECTS TO ACTIVE ERROR
- Keeping "du" in du-imperative: "Lies du das Buch" → WRONG → "Lies das Buch"
- Verbs ending in -ten/-den keep -e: "öffne! / arbeite! / antworte!" (not "öffn!")
- "wunderschön lecker" → wrong combination → "wunderbar lecker" OR just "wunderschön"
