**CATEGORY: seit — Interrogative Phrase Structure**

```
Wrong:   Seit wann wohnt sie in diese Stadt (1.2 item 8)
Correct: Seit wie vielen Jahren wohnt sie hier?
Pattern: "Seit wann" asks since when (a point in time); "seit wie vielen Jahren" asks for duration — the prompt requires duration; also in diese Stadt → in dieser Stadt (dative after in for location).

Wrong:   Wie lange arbeiten Sie hier (1.2 item 7)
Correct: Wie lange arbeiten Sie schon hier?
Pattern: schon is required to signal ongoing duration with present tense.

Wrong:   Seit wann wohnen Sie hier (Block 3 item 13)
Correct: Seit wie vielen Jahren wohnen Sie hier?
Pattern: Same as 1.2 item 8 — seit wann asks for a point, seit wie vielen Jahren asks for a count.
```
