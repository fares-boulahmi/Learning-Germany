### ❌ Errors — This Set

**CATEGORY: Word Order — Verb in Position 2**

```
Wrong:   Das Kind seit drei Stunden schläft (Dative drill item 5)
Correct: Das Kind schläft seit drei Stunden
Pattern: The finite verb must occupy position 2; "seit drei Stunden" is an adverbial, not the predicate.

Wrong:   Wie lange du arbeitet hier? (Speed Production item 13)
Correct: Wie lange arbeiten Sie schon hier?
Pattern: In a question beginning with an interrogative, the verb precedes the subject (V2 inversion); also wrong register (du instead of Sie).

Wrong:   Seit wann Jahren sie wohnt hier? (Speed Production item 20)
Correct: Seit wie vielen Jahren wohnt sie hier?
Pattern: V2 inversion is mandatory in questions; "Seit wann Jahren" is not a valid phrase.
```

**CATEGORY: Article — Incorrect or Missing**

```
Wrong:   Das Waschmaschine ist neue (2.1 item 2)
Correct: Das ist eine neue Waschmaschine
Pattern: die Waschmaschine is feminine — "Das" as a subject referring to it is incorrect; the restructured sentence requires "eine."

Wrong:   Das ist Stühle (2.2 item 2)
Correct: Das sind Stühle
Pattern: "Das sind" is required with a plural predicate noun.

Wrong:   Das ist Ihr Kleiderschrank, oder ist euer Kleiderschrank? (2.2 item 3)
Correct: Ist das Ihr Schrank, oder ist das ihr Schrank?
Pattern: The sentence is a question; "ist das" is required; "euer" is the du-plural possessive — wrong register (formal Sie required).

Wrong:   Ich sehe ein Kühlschrank (2.2 item 9/10 — listed as 9 in student output)
Correct: Ich sehe einen Kühlschrank
Pattern: Kühlschrank is masculine; accusative masculine requires -en ending (einen).

Wrong:   Wir haben keine Spüle im das Zimmer (Speed Production item 10)
Correct: Wir haben keine Spüle in diesem Zimmer
Pattern: "im" = in + dem (contracted); "im das" is ungrammatical — requires "in diesem."

Wrong:   Eure Kinder schläft seit drei Stunden (Speed Production item 15)
Correct: Ihre Kinder schlafen seit drei Stunden
Pattern: "eure" is the du-plural possessive — wrong register; plural subject "Kinder" requires "schlafen," not "schläft."

Wrong:   Die Waschmaschine meines Großvater kostet 500 Euro (Speed Production item 18)
Correct: Die Waschmaschine meines Großvaters kostet 500 Euro
Pattern: Genitive singular masculine requires -s/-es ending on the noun (Großvaters).
```
