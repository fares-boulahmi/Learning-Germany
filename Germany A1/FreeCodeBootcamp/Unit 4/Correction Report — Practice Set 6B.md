### 🔴 Active Errors — Require attention in Set 7

**1. Verb pair confusion: stehen/stellen, liegen/legen** This is the most important remaining error. Fares consistently confuses paired verbs:

- "liegen" (to be lying — position) vs "legen" (to lay/put — action)
- "stehen" (to be standing — position) vs "stellen" (to place/stand up — action)
- "sitzen" (to sit) — incorrectly used for furniture in E.1 and E.3: "Der Kleiderschrank sitze" — furniture does not "sit", it "stands" (steht).

This error appeared in B.3 (×3), E.1, and E.3. It is consistent and systematic.

**2. Accusative plural noun form: "die Stühle" not "die Stühlen"** In accusative plural: article = "die", noun = no change (die Stühle). In dative plural: article = "den", noun = add -n (den Stühlen). He adds -n to the noun even in accusative plural: "zwischen die **Stühlen**" (wrong).

**3. Spelling — three words that have been wrong since Set 6:**

- "über" written as "uber" (missing umlaut ü)
- "Sommer" written as "Summer" (English spelling)
- "Berge" written as "Barge" (wrong vowel)

**4. Furniture verb: "stehen" not "sitzen"** Furniture stands (steht) — it does not sit (sitzt). "Der Kleiderschrank sitzt" is wrong. "Der Kleiderschrank **steht**" is correct.

**5. "in + das/dem" confusion in free writing**

- "ich bin in **das** Badezimmer" — "bin" = location → dative → "**im** Badezimmer"
- "Ich hänge die Lampe **in** die Wand" — wrong preposition; hanging on a wall = "**an** die Wand"

**6. Missing words under time pressure** In E.3: dropped "September" from birthday sentence, dropped "Geburtstag" from birthday sentence. These omissions suggest attention to completeness, not grammar gaps.

## Curriculum Question: When to Make Set 7?

**Recommendation: Complete lessons 5.1–5.12 first, then generate Set 7 for all of them together.**

**Reasoning:**

Lessons 5.1–5.7 cover: time (Wie spät ist es?, clock times, half/quarter past/to), and daily routine morning vocabulary (I get up, I shower, I have breakfast, I go to work). These are closely connected topics — time expressions are needed to describe the daily routine ("Ich stehe um sieben Uhr auf"), and the daily routine verbs appear across multiple lessons (5.5, 5.6, 5.7 and likely 5.8–5.12).

If a set is made from only 5.1–5.7, it will be incomplete — you will not yet have afternoon/evening routines, likely not the full set of daily activity verbs, and the time expressions will not have been practised in the full range of contexts they appear in once 5.8–5.12 are done.

A single Set 7 covering 5.1–5.12 will be richer, more varied, and allow proper integration of time + routine vocabulary in dialogues and descriptions. It will also allow Set 7 to carry forward the unresolved errors from Sets 5, 6, and 6B (verb pairs stehen/stellen/liegen/legen, spelling fixes, accusative plural noun forms).

**In short: finish 5.1–5.12, then send the complete lesson list for Set 7 generation.**
