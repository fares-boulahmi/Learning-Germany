## Part 2: Target B — Dual-Case Prepositions in Production

### Exercise B.1: Location or Movement?

1. Die Katze liegt auf dem Sofa. → **L** (liegt = is lying = position)
2. Ich lege das Buch auf den Tisch. → **M** (lege = I lay/put = action)
3. Das Buch liegt auf dem Tisch. → **L** (liegt = is lying = position)
4. Er hängt das Bild an die Wand. → **M** (hängt = he is hanging it up = action)
5. Das Bild hängt an der Wand. → **L** (hängt = it is hanging there = position)
6. Wir fahren in die Berge. → **M** (fahren = to travel, movement)
7. Wir wohnen in den Bergen. → **L** (wohnen = to live, position)
8. Sie stellt den Stuhl neben den Tisch. → **M** (stellt = puts/places = action)
9. Der Stuhl steht neben dem Tisch. → **L** (steht = stands/is standing = position)
10. Ich gehe in die Küche. → **M** (gehe = I go, movement)
11. Ich bin in der Küche. → **L** (bin = I am, position)
12. Er legt die Katze auf das Bett. → **M** (legt = puts/lays = action)
13. Die Katze schläft auf dem Bett. → **L** (schläft = sleeps = position)
14. Sie hängt die Lampe über den Tisch. → **M** (hängt = hangs it up = action)
15. Die Lampe hängt über dem Tisch. → **L** (hängt = is hanging = position)

> **Position/state verbs (dative):** liegen, stehen, hängen (be hanging), schlafen, wohnen, sein, sitzen **Action/movement verbs (accusative):** legen, stellen, hängen (to hang up), gehen, fahren, kommen (movement toward)

## Quick Reference Summary

### The Two Ordinal Forms

|Context|Rule|Example|
|---|---|---|
|After **am** (birthday, date)|**-ten / -sten**|am **dreißigsten** Mai|
|After **Heute/Morgen/Gestern ist/war der**|**-te / -ste**|der **dreißigste** Mai|

### Dual-Case: Position vs Movement

|Type|Case|Key verbs|Article (masc)|
|---|---|---|---|
|WHERE something IS|**Dative**|ist, liegt, steht, hängt, schläft, wohnt|**dem**|
|WHERE something GOES/IS PLACED|**Accusative**|geht, fährt, legt, stellt, hängt (up)|**den**|

### Attributive Adjective Endings (after ein/eine/einen)

|Position|Ending|
|---|---|
|Nominative/Accusative feminine|**-e** (eine alte Kommode)|
|Accusative masculine|**-en** (einen alten Tisch)|
|Nominative/Accusative neuter|**-es** (ein altes Bett)|
|ALL dative|**-en** (einem alten Tisch / einer alten Kommode)|
|Plural (nom/acc)|**-e** (braune Haare)|

### Plural Dative

**ALWAYS "den"** + noun usually adds -n: "den Stühlen", "den Tischen", "den Wänden"
